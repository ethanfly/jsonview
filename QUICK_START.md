# 快速启动指南

## 应用优化后的启动流程

### 1. 开发模式
```bash
npm run tauri:dev
```

### 2. 构建生产版本
```bash
npm run build
```

## 优化效果对比

### 优化前
- ❌ 启动时白屏 500-1000ms
- ❌ 主题切换有闪烁
- ❌ 等待 Google Fonts 加载
- ❌ 窗口立即显示但内容未就绪

### 优化后
- ✅ 骨架屏立即显示，无白屏
- ✅ 主题预加载，无闪烁
- ✅ 使用系统字体，无网络延迟
- ✅ 窗口在内容准备好后才显示

## 关键优化点

1. **骨架屏**: 内联在 HTML 中，立即渲染
2. **窗口控制**: 初始隐藏，100ms 后显示
3. **主题预设**: Vue 挂载前就设置主题
4. **延迟初始化**: 非关键逻辑延迟 50ms
5. **系统字体**: 无需加载外部字体
6. **代码分割**: Vue/Tauri 依赖分离

## 性能指标

- **首次内容绘制 (FCP)**: ~150ms
- **可交互时间 (TTI)**: ~400ms
- **完全加载**: ~500ms

## 注意事项

- 骨架屏会在应用挂载后自动移除
- 窗口显示延迟可在 `src-tauri/src/main.rs` 中调整
- 主题设置在 `src/main.ts` 中预加载
